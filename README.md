# Noir - Salão Unissex

Projeto de um site para um salão de beleza unissex, com foco em um design moderno, elegante e minimalista.

## Sobre

O **Noir** é um site desenvolvido para representar um salão de beleza, trazendo uma identidade visual sofisticada e experiência agradável para o usuário.

## Tecnologias

* HTML
* CSS
* JavaScript

## Funcionalidades

* Layout responsivo
* Design moderno e minimalista
* Seções organizadas (serviços, galeria, contato, etc.)
* Interface visual atrativa

## Estrutura

```
/assets
/css
/js
index.html
```

## Como rodar

1. Clone o repositório:

```bash
git clone https://github.com/pgasaabyy/Noir---Salao-Unissex.git
```

2. Abra o arquivo `index.html` no navegador
